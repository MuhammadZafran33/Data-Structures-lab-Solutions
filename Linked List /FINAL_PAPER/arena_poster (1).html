üéØ The Problem
Build the matchmaking and leaderboard engine for "Arena of Ratings", a competitive game. Each player has a unique rating, and the system must quickly answer queries such as:

‚úì Find player by rating
‚úì List players in rating interval
‚úì Find nearest opponent
‚úì Find next higher/previous lower player
‚úì Compute rank and k-th player
‚úì Calculate distance in tree structure

The ordering must be based exclusively on player rating (unique integer).
×
üí° The Solution
Data Structure: Custom AVL Tree

AVL (Adelson-Velsky and Landis) Tree is a self-balancing binary search tree that maintains height balance through rotations. Key features:

‚úì O(log n) insertion, deletion, search
‚úì Automatic rebalancing via rotations
‚úì Perfect for range queries
‚úì Supports successor/predecessor queries

Implementation: Pointer-based with height and subtree size tracking for efficient queries.
